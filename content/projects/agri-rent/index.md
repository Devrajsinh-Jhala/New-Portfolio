## About

Agri Rent was developed during a hackathon as a functional web store for renting agricultural equipment and machines. It includes a commerce flow where product details are managed through Sanity CMS.

The platform also includes payment handling with Stripe and authentication through Clerk, making the rental journey more complete from browsing to checkout.

## Features

- Functional product pages
- Authentication and user management
- Payments handled by Stripe
- Product details managed by Sanity CMS
- Redux-based state management

## Tech Stack

- Frontend: Next.js, Tailwind CSS, JavaScript
- Backend and content: Sanity CMS
- State management: Redux
- Payments: Stripe

## What I Learned

This project gave me practical experience building a commerce-style workflow under hackathon constraints. It helped me connect product data, authentication, state management, and payment flows into a single application.
