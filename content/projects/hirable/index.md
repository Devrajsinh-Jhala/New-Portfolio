## About

Hirable is a frontend project for an onboarding and hiring platform. It includes separate user-facing routes for colleges, students, companies, login, signup, and email verification.

The project focused on building a structured multi-page frontend with consistent styling and clear navigation across different user roles.

## Sitemap

- Home page
- College page
- Students page
- Companies page
- Login page
- Sign up page
- Email verification page

## Tech Stack

- React
- Redux
- Tailwind CSS

## What I Learned

Hirable helped me practice route planning, frontend state management, responsive UI structure, and the basics of turning a multi-screen idea into a navigable product prototype.
